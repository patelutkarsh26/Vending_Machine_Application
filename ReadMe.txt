This is a application developed in C++ for Vending Machine.

When this application is executed it will ask user for total amount entered in terms of Quarters and then it will display it in Dollars.
After that it will ask to select type of item which user wants and then it will show list of items particularly of that type along with its price. Based on the users choice it will undergo vending process and also deducts amount from total balance (which is entered at the beginning). User can continue to purchase or can exit from application. At the end application will display total amount entered, total amount spent and change.
